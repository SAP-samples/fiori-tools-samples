## Description 

This is a sample app to manage products. This app is based on a SAP Fiori elements for OData V2 list report object page project. 

For more information on how to develop this application, see [Create a List Report Object Page App with SAP Fiori Tools](https://developers.sap.com/group.fiori-tools-lrop.html).

## Requirements

This app is intended as a sample app that can be previwed using SAP Fiori tools in SAP Business Application Studio or Visual Studio Code. For more information on how to get started with SAP Fiori tools, see [Getting Started with SAP Fiori Tools](https://help.sap.com/docs/SAP_FIORI_tools/17d50220bcd848aa854c9c182d65b699/2d8b1cb11f6541e5ab16f05461c64201.html).

You must have access to the SAP Gateway Demo system (ES5) for this sample app. For more information on how to create an account if you don't already have one, see [Create an Account on the SAP Gateway Demo System](https://developers.sap.com/tutorials/gateway-demo-signup.html).

## Download and Installation

1. After cloning the repository, right-click on the app folder (such as `myfioriapp`) and click "Open Integrated Terminal".
2. In the terminal, type `npm install` and press Enter.
3. Click the "SAP Fiori" icon on the activity bar.
4. Right-click the app name in the "Application Modeler" view to open the "Application Info" page.
5. Review and perform any suggested actions under `Application Status`.
6. Right-click the app name in the `Appication Modeler` view, click `Preview Application` and click `start` to preview the application with live data from the back end. Alternatively, you can click `start-mock` to preview with mock data.
