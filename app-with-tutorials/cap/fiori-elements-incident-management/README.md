## Description 

This is a sample app to manage incidents. This app is based on a SAP Fiori elements for OData V4 list report object page. 

For more information on how to build this app, see [Create an Incident Management App with SAP Fiori Elements for OData V4](https://developers.sap.com/group.fiori-tools-odata-v4-incident.html).

## Requirements

This app is intended as a sample app which can be previewed using SAP Fiori tools in SAP Business Application Studio or Visual Studio Code. For more information on how to get started with SAP Fiori tools, see [Getting Started with SAP Fiori Tools](https://help.sap.com/docs/SAP_FIORI_tools/17d50220bcd848aa854c9c182d65b699/2d8b1cb11f6541e5ab16f05461c64201.html).

## Download and Installation

1. After cloning the repository, open the `fiori-elements-incidents-management` folder in the root of your workspace.
2. At the root of the workspace, type `npm install` in the terminal.
3. Click on "SAP Fiori" in the activity bar.
4. Right-click on your project and click `Preview Application`.
